### ⚠️ 数据格式
文件使用 `|` 分隔字段，作者用 `;` 分隔：
```
1001|Title|Author1;Author2|Kw1|Kw2|...|Year|Citations|Abstract
```

### ⚠️ 编译顺序
确保先实现Paper和FileManager，其他模块才能正常工作：
```
Paper.cpp → FileManager.cpp → main.cpp → 其他模块
```

### ⚠️ 测试策略
